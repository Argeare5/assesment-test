# DeFi Real Estate - Take-Home Assessment

Thank you for your interest in joining our team.

This short assessment is designed to evaluate your React skill.

**Expected completion time: ~30 minutes**

## Setup

```bash
nvm install 20
nvm use 20

npm install
npm start
```

The application will be available at:

```text
http://localhost:3000
```

## Tasks

### 1. Complete Wallet Connection

The **Connect Wallet** button is currently incomplete.

Please implement:

* Wallet connection using MetaMask
* Display the connected wallet address
* Handle account changes
* Basic error handling (e.g. MetaMask not installed, user rejects connection)

### 2. Fix Homepage Responsiveness

Review the homepage and fix any responsiveness issues.

Ensure the application works properly on:

* Mobile
* Tablet
* Desktop

## Submission

Once you have completed the assessment:

1. Push your solution to a **public GitHub repository**
2. Share the **repository URL** with us
3. Include a short summary of:

   * What you changed
   * Any assumptions you made
